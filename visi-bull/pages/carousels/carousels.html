Carousels

-- --, 2014 {% endblock %} {% block content %}

Each carousel consists of:
A container in which the items are placed (and on which the effects like dragging operate).
A set of items inside the container (each of which records its own width, horizontal padding and height).
The effects applied to allow the carousel to move properly.
The carousel does not care about the shape of the items, its just places and manipulates them.

With centered items it is often best to manually define the carousel size (otherwise can't tell that it is centered).


Single Item Carousels

Non-infinite Scrolling

var svg = d3.select(...).append("svg");
var items = ...  // Create the items.
var carousel = carouselCreator(items)
    ...
    .isInfinite(false)
    .itemsToShow(1)
    .itemsToScrollBy(1)
    ...
svg.call(carousel);
